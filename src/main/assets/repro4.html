RISIKO DISFUNGSI SEKSUAL   

DEFINISI
Berisiko mengalami perubahan fungsi seksual selama fase respon seksual berupa hasrat, ternagsang, orgasme, dan relaksasi yang dipandang tidak memuaskan, tidak bermakna/tidak adekuat

  2. Gangguan urologi
  3. Gangguan endokrin
  4. Keganasan
  5. Faktor ginekologi (mis/ kehamilam, pasca persalinan)
  6. Efek agen aframakologis

Psikologis

  2. Kecemasan
  3. Penganiayaan psikologis/seksual
  4. Penyalhgunaan obat/zat
